# crypto-lab
# Cryptography Lab Assignment

This repository contains Python implementations for basic cryptography concepts.

## Problem 1: Number Theory Foundations
- Prime number checking  
- GCD using Euclidean Algorithm  
- Co-prime checking  
- Euler's Phi Function  
- Fermat's Little Theorem  
- Euler's Theorem  
- Modular arithmetic operations  

File: Number Theory Foundations.py

## Problem 2: Algebraic Structures
- Verification of Group, Ring, and Field in Z_n  
- Closure, identity, and inverse properties  

File: Algebraic Structures.py

## How to Run

Make sure Python is installed.

Run the programs:

python problem1_number_theory.py  
python problem2_algebraic_structures.py
